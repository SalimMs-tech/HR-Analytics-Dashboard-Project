# HR-Analytics-Dashboard-Project
This Power BI dashboard analyzes employee salary, department performance, and workforce distribution across departments.
HR Analytics Dashboard
Overview
This project is an HR Analytics Dashboard created in Power BI to analyze employee data, department performance, and salary distribution.

Tools Used
Power BI
Excel
Dashboard Features
Employee Count KPI
Salary Analysis
Average Performance Analysis
Department Comparison
Interactive Department Filter
Key Insights
Finance department has the highest salary allocation.
HR department maintains strong employee performance.
IT department has fewer employees compared to other departments.
